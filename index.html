Introduction

The name of App is "BMI Calculator". This App takes body weight and height as input and calculate BMI.

Project requires:

  • Some form of input
  • Some operation on the ui input in server.R
  • Some reactive output displayed as a result of server calculations
  • You must include endough documentation.
  • The documentation should be at the Shiny website itself.

Widgets of "BMI Calculator"

This App includes 2 widgets:

  • sliderInput:
    • First slider is to choose body weight (lb)
    • Second slider is to choose height (ft)
  • submitButton: This button submits body weight and height to server for calculation.
